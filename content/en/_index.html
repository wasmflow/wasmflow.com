Wasmflow is a free and open platform for building applications out of WebAssembly code containers.

Wasmflow lets you write code and reuse it easily, everywhere. It's 100% Rust and takes ideas from Docker, Kubernetes, Erlang, Rx, FBP, plus more and brings them into one place.

Write code in any language and connect it to any other code with one tool.

Installation

Wasmflow has precompiled binaries for popular platforms or you can compile from source.
